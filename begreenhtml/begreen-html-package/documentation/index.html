Begreen - Responsive Multi-Purpose HTML Template

Begreen is a modern, clean and professional HTML Template, It is fully responsive, it looks stunning on all types of screens and devices.

Template Features

  • HTML5 / CSS 3
  • Fully Responsive (Tested on Multiple Devices)
  • Full Width Layout
  • Shop Design
  • Multi-options for Home Page
  • Inner Pages: Products Grid 4 Columns, Products Grid 3 Columns, Products 3 Columns with Sidebar, Products 2 Columns with Sidebar, Product List, Simple ProductSALE OFF, Variable Product, Out Of Stock ProductOUT, Grouped Product, Affiliate/ External Product, Product Full-width, Recent Products, Best Selling Products, Sale Products, Top Rate Products, Shop, Cart, Checkout, My Account, Wishlist, Blog list, Blog Grid 2 columns, Blog Grid 3 columns, Detail Blog, Blog masonry, Blog Single post, About, Contact,...
  • Mega Top Menu
  • Mega Vertical Menu
  • Sticky Menu
  • Brand slide
  • Special Product Groups
  • Services
  • Carousel : Best Sellers, On Sales, New Products, Related Products, Testimonials, Categories, Filter Section, Banners, Slideshow,...
  • jQuery Enhanced
  • HTML files
  • HTML5 and CSS3 tableless design
  • List & Grid View (Products)
  • Compatible Browsers: IE9+, Firefox, Safari, Opera, Chrome
  • Included documentation.

How to put the template online

After you have downloaded the template to your liking by replacing the text and images with your own, you'll want to put it online but in order to be able to put your new template online there are some requirements you need to meet:

  1. Buy a domain name
  2. Buy a web hosting service
  3. Have a ftp client installed
